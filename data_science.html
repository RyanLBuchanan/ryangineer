Data Analytics Mathematics

Regression Analysis

An understandable equation to predict a single outcome based on multiple predictor variables
  • Multiple Linear Regression
    y i   =   β 0   +   β 1 x 1 i   +   β 2 x 2 i   +   β 3 x 3 i   +   ε i
  • Linear Regression with Linear Algebra
    • Matrix-Vector Multiplication Function
      [ y 1 y 2 ]   =   [ 1 x 11 x 12 x 13 1 x 21 x 22 x 23 ] [ β 0 β 1 β 2 β 3 ]   +   [ ε 1 ε 2 ]
        ↳ where:
            Vector of Outcomes for Each Case   =   [ y 1 y 2 ]
            Matrix of Scores for Each Case   =   [ 1 x 11 x 12 x 13 1 x 21 x 22 x 23 ]
            Vector of Regression Coefficients (y-intercept & slopes)   =   [ β 0 β 1 β 2 β 3 ]
            Vector of Error Terms   =   [ ε 1 ε 2 ]

Structured Query Language (SQL)

  • Basic Anatomy & Physiology of SELECT Query
    1. Operation:  Describes what is going to be done with SELECT keyword followed by the names of columns to combine with functions.
    2. Data:  FROM keyword followed by one ore more tables connected together with reserved keywords, indicating what data should be scanned for filtering, selection & calculation.
    3. Conditional:  Filters the data to only rows that meet a condition, usually indicated by WHERE keyword.
    4. Grouping:  Special clause that takes rows of a data source, assembles them together using a key created by a GROUP BY clause, then calculates a value using the values from all rows with the same value.
    5. Post-processing:  Takes the results of the data & formats them by sorting & limiting the data using keywords such as ORDER BY & LIMIT.
